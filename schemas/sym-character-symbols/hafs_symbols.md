## Cardinality Analysis

### 1. Diacritics

Each base letter carries **at most one** diacritic from the set {haraka, tanween, sukun}. These are mutually exclusive — a letter cannot have both a haraka and a sukun, or two harakaat, etc.

The sole exception is **Shadda**, always co-occurs with exactly one haraka or tanween — it never appears alone. This raises the maximum combining count to **two** diacritics per letter, where one is always shadda.

**Cardinality rule:** `0..1` diacritics per letter, or `2` when one is shadda.

### 2. Extensions

Each base letter carries **at most one** extension symbol. The four extensions (Dagger Alef, Small Waw, Small Yaa, Small High Yaa) are mutually exclusive — no letter ever carries two.

**Cardinality rule:** `0..1` extension per letter.

Maddah (U+0653, Tajweed category) can co-occur with extensions on the same letter:

### 3. Stop Signs

Each base letter carries **at most one** stop sign. One exception: 36:52 (مَّرۡقَدِنَاۜۗ) carries both Seen Sakt and Qili.

**Cardinality rule:** `0..1` stop sign per letter (one exception at 36:52).

### 4. Tajweed

Each base letter carries **at most one** tajweed symbol. The nine tajweed symbols are mutually exclusive — no letter ever carries two.

**Cardinality rule:** `0..1` tajweed symbol per letter.

### Cross-Category

All 6 category pairs co-occur. No pair is mutually exclusive. A single base letter can carry symbols from all 4 categories simultaneously (73 instances).
